# Fine Tuning (Otimização)

Durante a criação de um modelo de inteligencia artificial podemos utilizar vários parâmetros para determinar como o algoritmo busca por padrões para realizar novas previsões. Ao utilizar diferentes parâmetros ou até mesmo combinações de parâmetros podemos obter modelos com diferentes desempenhos.

O processo de fine tuning consiste em ajustar os parâmetros fornecidos para os modelos de forma que possamos encontrar aqueles que resultam nos melhores resultados para os dados que estamos utilizando.
