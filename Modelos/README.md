# Modelos de Inteligencia Artificial

Modelos de Inteligencia articial são algoritmos dedicados a reconhecer padrões a partir de conjuntos de dados (datasets) para realizar tarefas que normalmente exigem inteligencia humana. Esses algoritmos normalmente implementam calculos matemáticos que podem ser utilizados para resolver problemas especificos e por esse motivo existem diferentes tipos de modelos de Inteligencia Artificial, cada um dedicado a realizar uma tarefa especifica e de uma forma especifica.

Essa variedade de algoritmos faz com que seja necessário entender como esses modelos funcionam e a natureza dos dados com que podem lidar. Assim é possivel selecioanr o modelo ideal para realizar uma tarefa, além dos ajustes necessários para otimizar o desempenho desses modelos.

Ainda é necessário destacar que existem duas formas de aprendizado, são essas:

- **Aprendizado Supervisionado:** Nessa técnica o algoritmo aprende à partir de dados rotulados, isso é, o modelo recebe um conjunto de entradas (features) e o target (saída) esperada  saída esperada. Dessa forma é possivel realizar previsões e classificações precisas.
  
- **Aprendizado Não Supervisionado:** Nessa técnica o algoritmo recebe apenas dados não rotulados, isso é, o modelo tem acesso apenas as features mas não possui nenhum target associado. O objetivo desses modelos é identificar padrões e estruturas ocultas nos dados, podendo ser utilizado para realizar analise de dados.

# Dataset de teste

Como já detalhado anteriormente, modelos de Inteligencia Artificial variam em questões de aplicação e formas de aprendizado. Portanto, para facilitar o entendimento dos modelos apresentados cada um vai lidar com um conjunto de dados que melhor se adeque aos seus casos de uso.

Os dados pré-definidos podem ser encontrados na pasta [Data](../Data/) desse repositório, porém, em algums casos será utilizado um bloco de código dedicado a gerar dados que melhor se adequam ao contexto apresentado.
