# Modelos de Inteligencia Artificial

Modelos de Inteligencia Artificial são algoritmos dedicados a buscar e aprender padrões em conjuntos de dados com o objetivo de realizar previsões à partir das informações aprendidas.

É importante ressaltar que, existem diferentes modelos de Inteligencia Artificial, sendo que, cada um possui sua própria forma de aprender, e em algus casos, possuem até mesmo finalidades diferentes. Dessa forma, é necessário compreender que cada tipo de modelo deve ser empregado em contextos especificos, sendo que, cada modelo pode ainda apresentar diferentes formas de apredizado, que podem apresentar melhores desepenhos de acordo com o contexto dos dados.

# Dataset de teste

Como ja detalhado anteriormente, modelos de Inteligencia Artificial variam em questões de aplicação e formas de aprendizado. Dessa forma, para cada modelo são necessários conjuntos de dados diferentes para uma melhor ilustração dos cenários de uso que cada uma apresenta.

Dessa forma, para cada modelo apresentado nesse diretório será utilizado um conjunto de dados diferentes. Os dados pré-definidos podem ser encontrados na pasta [Data](../Data/) desse repositório, porém, em algums casos será utilizado um bloco de código dedicado a gerar dados que melhor se adequam ao contexto apresentado.
