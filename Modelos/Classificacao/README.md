# Modelos de Regressão

Modelos de Classificação são usados para estimar uma classe ou categória. Por exemplo:

- Diagnósticos Médicos
- Reconhecimento de Imagem
- Previsão de Crédito

Ainda é possivel agrupar as previsões realizadas pelos modelos de Classificação da seguinte forma:

- Classificação Binária -> Apenas duas classes (0/1, spam/não spam, câncer/não câncer)
- Classificação Multiclasse -> Mais de duas classes (digitos de 0 a 9)
- Classificação Multilabel -> Uma mesma amostra pode ter mais de um rótulo (Tags em um artigo)
